# Security policy

## Supported versions

Security fixes are applied to the latest published version of each `@uniswap-widget/*` package.
Older versions are not guaranteed to receive patches.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability. Use GitHub's private vulnerability
reporting form:

https://github.com/houtanrocky/uniswap-widget/security/advisories/new

Include the affected package and version, impact, reproduction steps, and any suggested mitigation.
You should receive an acknowledgement within seven days. Details will remain private until a fix or
coordinated disclosure is ready.

## Scope and risk

This project is an independent community project and is not affiliated with or endorsed by Uniswap
Labs or the Uniswap Foundation. It has not been independently audited. Integrators are responsible
for reviewing contract interactions, token approvals, RPC and proxy configuration, wallet behavior,
and the versions of all peer dependencies before handling real assets.
